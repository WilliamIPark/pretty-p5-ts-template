# Pretty P5 TypeScript
A very basic template to set up a TypeScript environment for P5.js using Instance Mode.

![P5 _ Typescript](https://github.com/WilliamIPark/pretty-p5-typescript/assets/445688/4d681a23-17cf-459e-a851-ed51e9bf4b52)

## Installation
- Pull this repo down.
- Run `npm install` to download the dependancies.
- Run `npm run dev` to load up the webpage.

## Instructions
- Start in `src/main.ts`, where you will find some sample P5 content.
- Modify page styles in `src/style.css`
